Multi-Object Tracking (MOT) is a fundamental computer vision challenge that requires tracking multiple objects simultaneously throughout a video sequence. This is essential for applications like autonomous driving, surveillance systems, and sports analytics.

My research mission is to extend human perception through intelligent visual tracking systems ‚Äî enabling machines to see, reason, and remain aware where human attention cannot.

Occlusions

Objects disappear temporarily behind others. While the human visual system can track through occlusions, most algorithms struggle when objects are not visible.

ID Switches

Losing track of identities after occlusion. For example, when two similar people cross paths and are later confused with each other.

Similar Appearances

Distinguishing between visually similar objects. Particularly challenging in uniform scenarios (e.g., sports teams, siblings).

Crowded Scenes

Complex interactions between multiple objects create frequent occlusions and are computationally demanding to process.

Small Data Domains

Limited training examples in specialized areas like medical applications and infant monitoring require methods that can learn from sparse data.

Research Vision

Advancing visual tracking to enable reliable machine perception in dynamic, complex, and attention-limited environments.

Healthcare

Enabling reliable infant tracking and sleep monitoring in medical environments where human attention is limited.

Safety

Improving vehicle-pedestrian interaction understanding for autonomous driving and traffic monitoring systems.

Sports Analytics

Enabling detailed player tracking and team formation analysis without requiring wearable sensors.

DragonTrack: Transformer-Enhanced Graphical Multi-Person Tracking in Complex Scenarios

Published in WACV 2025

DragonTrack introduces a novel approach to multi-object tracking by combining transformer-based detection with graph modeling for inter-object relationships. Our framework achieves state-of-the-art performance on standard benchmarks while maintaining efficient computational requirements.

Key innovations:

  • Transformer-based detection for spatial and appearance features
  • Graph modeling for inter-object relationships
  • Robust identity preservation in crowded scenes

Technical Approach

Key Innovations

Transformer-based Detection (DETR)

Utilizes spatial and appearance features for robust object detection even in challenging scenarios

Graph Convolutional Network (GCN)

Models inter-object relationships to maintain identity consistency during interactions

Learnable Sinkhorn Assignment

Provides optimal track-detection matching for maintaining identity consistency

Contrastive Loss + Weighted BCE

Creates identity-preserving embeddings that distinguish between similar-looking objects

Performance Evaluation

DragonTrack was evaluated against both state-of-the-art CNN-based and Transformer-based methods on the MOT17 and DanceTrack datasets.

MOTE: More Than Meets the Eye

Optical Flow-Based Multi-Object Tracking with Prolonged Occlusion Handling

Accepted at International Conference on Machine Learning (ICML), 2025

MOTE builds upon traditional tracking frameworks by incorporating optical flow and softmax splatting for disocclusion features. This approach significantly reduces identity switches in scenes with prolonged occlusions.

Key contributions:

  • Optical flow estimation for tracking through occlusions
  • Softmax splatting for disocclusion-aware representations
  • Enhanced track embedding module for identity continuity

How MOTE Sees the World

MOTE processes video frames (left), estimates optical flow (middle), and generates disocclusion matrices (right) to track objects through occlusions.

Technical Approach

Key Components

Optical Flow Estimation

Computes dense motion fields between frames to model object movement‚Äîeven when partially or fully occluded.

Softmax Splatting Module

Warps feature maps along flow vectors to create disocclusion-aware representations that maintain object identity through occlusions.

Enhanced Track Embedding Module (ETEM)

  • Inputs: Query features, positions, and splatted motion features
  • Uses multi-head attention to fuse temporal and appearance information
  • Applies linear projection, add & norm, and feed-forward layers
  • Outputs: Enhanced track query with improved identity continuity

Performance Evaluation

MOTE was evaluated on the MOT15, MOT17, MOT20, DanceTrack, and SportsMOT datasets, with particular focus on occlusion scenarios.

UniTrack: A Differentiable Graph-Based Loss for Robust Multi-Object Tracking

Under Review

UniTrack presents a unified tracking framework with a graph-based differentiable loss function that can be integrated with existing tracking architectures. This approach eliminates the need for scenario-specific tracking systems.

UniTrack delivers the following advantages:

  • Unified loss function for tracking optimization
  • Seamless integration with existing architectures
  • Addresses post-occlusion, temporal, and cross-subject errors

Technical Approach

Graph-Based Formulation

Graph Definition:

  • Nodes (Vt): Represent tracked objects at time t
  • Edges (Et): Encode spatial and temporal relationships
  • Weights (Wt): Edge weights derived from learned flow variables

Unified Loss Formulation:

  • Temporal Edges: Address Type 2 errors (temporal drift)
  • Spatial Edges: Address Type 3 errors (similar subjects)
  • Flow Conservation Constraint: Maintains track life cycle

Figure: UniTrack's graph-based architecture showing node connections and attention mechanisms

Performance Evaluation

UniTrack was tested with multiple tracking frameworks: MOTR, Trackformer, GTR, ByteTrack, MOTE and FairMOT. The enhanced algorithms were evaluated on the MOT17, MOT20, SportsMOT and DanceTrack datasets.
